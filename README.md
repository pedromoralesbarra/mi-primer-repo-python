# Mi primer repositorio Python

Este es un proyecto simple para aprender el flujo basico de Git y GitHub con Python.

## Que hace

Ejecuta un programa en Python que pide tu nombre y muestra un saludo.

## Como ejecutarlo

```powershell
python main.py
```

## Comandos importantes

Ver el estado del repositorio:

```powershell
git status
```

Guardar cambios en Git:

```powershell
git add .
git commit -m "Describe tu cambio"
```

Subir cambios a GitHub:

```powershell
git push
```
